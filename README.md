# project-backend

HOW TO START:

install python3

create venv

install packages from requirements.txt

USAGE:

in main directory:

flask --app hello run

/ endpoint prints simple text: Hello World!

/hello endpoint is a form with name. Enter it and You will get personalized greetings :)
